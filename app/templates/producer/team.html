{#} Template de gestión de equipo para productores en Gem-AvatART. Este template implementa la interfaz completa para que los productores gestionen su equipo de subproductores y usuarios finales. Incluye funcionalidades para invitar nuevos miembros, ver estadísticas detalladas de cada miembro, gestionar permisos y monitorear la actividad del equipo. El template incluye: - Lista completa de miembros del equipo con filtros - Estadísticas individuales de cada miembro - Funcionalidad para invitar nuevos miembros - Gestión de roles y permisos - Historial de actividad de cada miembro - Sistema de búsqueda y filtrado - Acciones administrativas (activar/desactivar usuarios) Clases Bootstrap utilizadas: - Layout: container-fluid, row, col-*, d-flex, justify-content-between - Cards: card, card-body, card-header -
