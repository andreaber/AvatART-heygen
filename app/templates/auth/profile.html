{# Template de perfil de usuario para Gem-AvatART. Este template implementa la página completa de gestión de perfil personal donde los usuarios pueden editar su información personal, cambiar avatar y visualizar datos del sistema. Incluye formularios diferenciados por rol y previsualización de avatar en tiempo real. El template incluye: - Formulario completo de edición de perfil con campos HTML estándar - Sistema de subida y previsualización de avatar con JavaScript - Sección de información comercial para productores -
